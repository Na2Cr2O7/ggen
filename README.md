# ggen
自动生草机
将文字翻译多次后生成视频

## 安装
```
pip install -r requirements.txt
```
你还需要下载ollama或ArgosTranslate  模型[https://drive.google.com/drive/folders/11wxM3Ze7NCgOk_tdtRjwet10DmtvFu3i]   ，更多内容查看config.ini.
### 安装ArgosTranslate模型 `InstallArgosTranslateModels.py`

## 使用方法
```
gg.py 翻译次数 文件名称或文字
```


```
gg.py 10 input.txt
```
